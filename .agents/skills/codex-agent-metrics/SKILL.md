# Codex Agent Metrics

Use for retrospective data, not user-visible reporting.

## Fields

- Task type and size.
- Risk level.
- Files changed.
- Checks run.
- Corrections or rework.
- Confidence and doubt.
- Lessons or follow-ups.

Prefer persistent memory when available; otherwise append to the project memory fallback or research cache.

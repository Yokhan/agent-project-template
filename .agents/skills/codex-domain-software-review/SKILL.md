# Codex Domain Software Review

Read `.claude/skills/domain-software-review/SKILL.md` for the full evidence catalog when needed.

Focus on:

- Module boundaries and dependency direction.
- Simplicity versus premature abstraction.
- Testability and observability.
- Security and error handling at boundaries.
- Reuse without coupling.

Lead with defects and decision risks, not general commentary.

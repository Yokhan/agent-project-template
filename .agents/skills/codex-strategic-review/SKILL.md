# Codex Strategic Review

Read `.claude/skills/strategic-review/SKILL.md` for the full checklist when needed.

## Process

1. State the real objective.
2. Identify the constraint and success metric.
3. Check whether the plan optimizes the user outcome or local task completion.
4. Compare at least one alternative.
5. Call out irreversible choices and weak assumptions.
6. Recommend the next smallest valuable move.

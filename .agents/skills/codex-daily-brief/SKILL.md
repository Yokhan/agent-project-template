# Codex Daily Brief

## Process

1. Read `tasks/current.md`, recent git history, lessons, and research cache.
2. Summarize yesterday or last session without dumping logs.
3. List current state, blockers, and next priorities.
4. Separate verified facts from assumptions.
5. Keep the brief short and actionable.
